記入上の注意事項

必ずお読みください。

  1. 入力について
    ・入力は原則として勤務した日のうちに行ってください。
    ・勤務日の翌月2日午前9時を過ぎて入力されたものは、支給が勤務日の翌々月となる場合があります。
  2. 氏名入力について
    ・氏名はフルネーム（例：山田太郎）で入力してください。
    ・姓と名の間にスペースは入れないでください。
  3. 日付入力について
    ・実施日を選択してください。
  4. 勤務時間入力について
    ・勤務開始・終了時刻は、実際に業務を行った時刻を記入してください。
    ・時刻入力後、総勤務分数が自動計算されるので確認してください。
  5. 休憩時間入力について
    ・取得した場合は開始・終了時刻を記入してください。
    ・休憩を取っていない場合は空欄のままで構いません（0分になります）。
  6. 勤務場所選択について
    ・当日実際に勤務した場所を選択してください（例：四条烏丸校）。
    ・在宅勤務の場合は「在宅リモート」を選択。
    ・交通費が異なる場合は特別交通費申請フォームで申請してください。
  7. 業務種別・依頼部署について
    ・主な業務内容に合わせて選択してください。
    ・依頼部署と指示担当者の記載ルール：
    • 被研修（合同）：講師部 / 研修担当者名
    • 被研修（各科研修）：自身の所属部署 / 研修担当者名
    • テスト監督（高卒）：高卒本部 / 大迫忠史
    • テスト監督（現役）：現役本部 / 佐々木朗洋
    • 依頼事務（例：採点等）：(依頼部署) / (担当者)
    • 質問対応（※寺子屋でない）：(所属科) / (上長 ：例)野口剛）
    • 生徒キャンセル（待機時間分）：総合受付課 / 矢野佑佳
    • その他：所属科 / 上長
    ・不明点はお問い合わせフォームへ。
  8. 業務内容・備考について
    ・具体的に簡潔に記入（例：合同研修参加）
    ・特記事項がある場合は備考欄へ記載。
  9. 送信前の確認
    ・内容に誤りがないか確認してください。
  10. その他
    ・面談コメントやマンスリーは授業給に含まれます。
    ・システム不具合や入力不明点はお問い合わせフォームからご連絡ください。
